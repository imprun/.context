# STORY-19.7: 배포 이력 타임라인 위젯

## 1. 개요
**Epic**: EPIC-019 ProductPublish 배포
**제목**: 배포 이력 타임라인 위젯
**담당자**: AI Agent
**상태**: ✅ 완료
**우선순위**: P1

## 2. 목적
배포 상세 페이지에 배포 이력을 타임라인 형태로 표시한다.

## 3. 구현 상세

### 3.1. UI 와이어프레임
```
┌─ 배포 이력 ──────────────────────────────────────────────────┐
│                                                               │
│  ●───────────────────────────────────────────────────────    │
│  │                                                            │
│  ├─● 2025-01-20 14:30  PUBLISHED by admin                    │
│  │   "Production release v2.0"                                │
│  │                                                            │
│  ├─○ 2025-01-20 14:00  DRAFT created by admin                │
│  │   Initial deployment configuration                         │
│  │                                                            │
│  └───────────────────────────────────────────────────────    │
│                                                               │
└───────────────────────────────────────────────────────────────┘
```

### 3.2. 타임라인 이벤트 타입
- `created` - 배포 생성
- `published` - 배포 실행
- `withdrawn` - 배포 철회
- `updated` - 설정 변경

### 3.3. 컴포넌트
- `PublishTimeline` - 타임라인 컨테이너
- `TimelineEvent` - 개별 이벤트 아이템

### 3.4. 데이터
현재는 `created_at`, `published_at`, `withdrawn_at` 필드를 기반으로
간단한 타임라인을 구성. 향후 별도 이력 테이블이 필요할 수 있음.
